
# 표/그림 객체 인식 데이터셋
 본 데이터셋은 논문 및 연구보고서 등 학술문헌의 표와 그림을 자동으로 추출하기 위한 데이터셋이다.

## 데이터 포맷(JPG, TXT)
|유형|설명|
|--|--|
|JPG|표/그림이 존재하는 연구보고서의 페이지를 이미지 파일로 변환|
|TXT|각 페이지에 존재하는 표, 그림, 캡션의 레이블 및 좌표 정보를 포함|
